Basic Concept of Web Technology

Define HTML

HTML stands for Hypertext Markup Language. It is the standard markup language used to create web pages. HTML defines the structure and content of a webpage using various elements and tags.

Define Client-server model of communications

The client-server model is a computing architecture where clients request resources or services from servers. Servers are responsible for processing these requests and sending back the requested data.

Write down the name of different types of servers

Different types of servers are email servers, file servers, and database servers.

Define a Web Server

A web server is a software or hardware component that serves web content to clients over the HTTP or HTTPS protocol. It processes client requests and delivers webpages, images, and other resources.

Define Data Centers

Data centers are centralized facilities used to house and manage computer systems, servers and networking equipment. They are designed to store, process, and manage data and applications reliably and efficiently.

Why do we need a Uniform Resource Locator (URL)?

A Uniform Resource Locator (URL) is used to uniquely identify and locate resources on the web. It enables users to access webpages and resources by specifying the resource's address.

Give an example of Uniform Resource Locator (URL) and specify all components of a URL

An example of a URL is:
https://www.example.com:8080/path/to/resource?param1=value1&param2=value2

  • Scheme: https
  • Host: www.example.com
  • Port: 8080
  • Path: /path/to/resource
  • Query: param1=value1&param2=value2

What is the difference between a static and a dynamic web site?

A static website contains fixed content that does not change unless manually updated. A dynamic website, generates content on-the-fly in response to user interactions or data from a database, making it more interactive and adaptable.

Why do we need Domain Name System (DNS)?

The Domain Name System (DNS) is used to translate human-readable domain name into IP addresses that computers use to identify each other on the internet. It makes it easier for users to access websites without needing to remember numerical IP addresses.

Describe in detail the process of DNS address resolution?

The DNS resolution process involves several steps, including querying DNS servers, caching, and ultimately resolving a domain name to an IP address. This process ensures efficient and accurate mapping between domain names and IP addresses.

Give some examples of generic top-Level domains (gTLD)

Examples of gTLDs include .com, .org, .net, .info, .gov, and .edu .

Give some examples of country code top-Level domains (ccTLD)

Examples of ccTLDs include .uk (United Kingdom), .ca (Canada), .jp (Japan), .au (Australia), and .de (Germany).

What is the function (purpose) of Hypertext Transfer Protocol (HTTP)?

HTTP is the protocol used for transferring data over the World Wide Web. Its primary purpose is to enable communication between web clients and web servers, facilitating the retrieval and display of web content.

Describe how a single web page is displayed on a web browser.

When a user enters a URL in a web browser, the browser sends an HTTP request to the web server. The server processes the request, retrieves the requested webpage's HTML and associated resources, and sends them back to the browser. The browser then renders the HTML, displaying the webpage's content.

Give some examples of HTTP request methods.

Common HTTP request methods include GET, POST, PUT, DELETE and HEAD.
