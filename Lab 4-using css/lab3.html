What are the main benefits of using CSS?

CSS allows for separating the content and presentation of a webpage, enhancing flexibility, maintainability, and loading speed.

What are different parts of a CSS style rule?

A CSS style rule typically consists of a selector and a declaration block containing one or more declarations.

What is an element selector? Give an example.

An element selector targets all instances of a specific HTML element. Example: p { color: blue; } applies blue color to all paragraphs.

What is a class selector? Give an example.

A class selector targets HTML elements with a specific class attribute. Example: .highlight { background-color: yellow; } applies a yellow background to elements with class "highlight".

What is an id selector? Give an example.

An id selector targets a single HTML element with a specific id attribute. Example: #header { font-size: 24px; } sets the font size of the element with id "header" to 24 pixels.

What are contextual selectors in CSS? Give an example.

Contextual selectors target an element based on its position within the HTML hierarchy. Example: article p { font-style: italic; } applies italic style to paragraphs within the "article" element.

What is an attribute selector? Give an example.

An attribute selector targets elements based on the presence or value of their attributes. Example: input[type="text"] { border: 1px solid black; } styles text input elements.

What is a pseudo selector? Give an example.

Pseudo selectors select elements based on their state or position. Example: :hover { color: red; } changes the text color of an element when hovered over.

What are the three cascade principles used by browsers when style rules conflict?

Specificity, importance, and source order are the three cascade principles used to resolve conflicting styles in CSS.
